# My Tools

## BiliBili Downloader

Since `yt-dlp` doesn't work well with **BiliBili** for me, and because the platform splits its media into **DASH** *(Dynamic Adaptive Streaming over HTTP)* streams, I wanted to automate the download process

To solve this, I wrote a **lightweight script** that extracts direct links to the **audio**, **video**, and **cover art**, while automatically generating a **ready-to-use PowerShell command** to download and merge everything

![BiliBili.png](BiliBili.png)

#### How to Use

1. Open the **BiliBili** video you want to download
2. Open your browser console _(Press `Ctrl + Shift + I` or `F12`)_
3. Paste the contents of [**BiliBili.js**](https://raw.githubusercontent.com/RootTool0/Just-Vocaloid-Lost-Media/refs/heads/main/BiliBili.js) into the console and hit Enter
4. The script will output 3 direct links _(video, audio, and cover)_ along with a custom **PowerShell** command
5. Open **PowerShell** and paste the generated command
6. **PROFIT!** The high-quality **.jpg** and **.mp4** files will be saved directly to your current directory

###### _(Note: Occasionally, the cover art might fail to download automatically. If this happens, simply click the provided artwork link in the console, right-click the image, and choose "Save image as...")_
